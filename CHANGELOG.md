## [1.0.0] - 2025-01-07

### Added
- Initial release of Daily Planner
- Task management with multiple views (Inbox, Today, Upcoming, Anytime, Someday, Logbook)
- AI-powered task breakdown and daily planning
- Project and area management with progress tracking
- Tag system for task categorization
- Cross-platform sync via Supabase
- Offline support with automatic sync
- Dark mode theme toggle
- Drag and drop task reordering
- Smart task input with natural language parsing
- Recurring task support
- Subtask management
- Task search functionality
- Finance management (transactions, debts, budgets, recurring payments)
- Weekly review with AI insights
- PWA support for offline web usage
- Desktop app support via Tauri
- Comprehensive documentation
- CI/CD workflows for automated builds

### Tech Stack
- React 18 with TypeScript
- Vite for fast builds
- Tailwind CSS with custom Things 3-inspired design
- Radix UI components
- Supabase for backend
- Tauri for desktop apps
- OpenRouter API for AI features

### Platform Support
- Web (any modern browser)
- Windows (via Tauri)
- macOS (via Tauri)
- Linux (via Tauri)
